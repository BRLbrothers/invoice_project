# 🚀 InvoiceIN GST – AI Ultra

A **modern GST Invoice Generator Web Application** with a futuristic dashboard UI.
Built for **freelancers, developers, and small businesses** to generate professional invoices quickly.

---

# ✨ Features

### 🧾 Invoice System

* Create professional GST invoices
* Download invoice as **PDF**
* Automatic total & GST calculation
* Invoice number and due date management

### 🏢 Business Profile

* Upload company logo
* Save business details
* Auto-fill invoice fields

### 👥 Client CRM

* Store client information
* Quick client selection
* Manage customer records

### 📦 Product Manager

* Save frequently used products
* Auto-fill product price and GST

### 📊 Analytics Dashboard

* Track invoices
* Revenue insights
* Statistics and charts

### 📁 Draft System

* Save unfinished invoices
* Load drafts anytime

### 🤖 AI Tools

* AI email generator
* AI suggestions

### 🎨 Modern UI

* Futuristic neon grid background
* Dark mode interface
* Responsive design

# 🛠 Tech Stack

Frontend

* HTML5
* CSS3
* JavaScript (ES6)

Libraries

* **jsPDF** – PDF generation
* **jsPDF AutoTable** – invoice tables
* **Chart.js** – analytics charts
* **QRCode.js** – QR payment codes

⚙️ Installation

Open the project folder:

cd invoice-project

Run the app by opening:

index.html

No server or installation required.

# 📄 Generate Invoice

1. Enter business details
2. Add products/services
3. GST automatically calculated
4. Click **Download PDF**

The invoice will be generated instantly.
